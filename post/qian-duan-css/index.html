2019-06-07

CSS引用的三种方法(优先级)

内联式CSS设置

把css代码直接写在现有的html标签中 如：

<p style="color:red">这里文字是红色。</p>

嵌入式CSS设置

把css代码写在标签之间

<style type="text/css">
span{
color:red;
}
</style>

外部式CSS设置

把css代码写成一个单独的外部文件 在内使用引入

<link href="index.css" rel="stylesheet" type="text/css" />

优先级顺序

在三个方法权值相同的情况下，并且嵌入式在外部式前面调用 (内联式 > 嵌入式 > 外部式)
